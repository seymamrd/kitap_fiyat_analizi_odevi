# Kitap Fiyat Karşılaştırma Sistemi

Bu proje, farklı kitap satış sitelerindeki kitap fiyatlarını karşılaştırmak amacıyla geliştirilmiş basit bir web uygulamasıdır.

Proje kapsamında kitap adı, yayınevi ve fiyat bilgileri halka açık web sayfalarından **eğitim ve öğrenme amaçlı** olarak alınmaktadır. Ticari kullanım hedeflenmemektedir.

Veriler veritabanına kaydedilmekte, fiyat değişimleri grafikler ile görselleştirilmektedir.

## Kullanılan Teknolojiler
- Python
- Flask
- Selenium
- Beautifulsoup
- SQLite
- HTML, CSS, JavaScript
